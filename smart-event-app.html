Smart Event App

Project Information

  • Category: Mobile Application
  • Client: University Final Year Project
  • Project Date: 2025
  • Tech Stack: Flutter, Firebase, Dart

About the Project

The Smart Event App is a mobile application developed using Flutter with Firebase backend for real-time event management, notifications, and RSVP tracking. It was created as a university final year project.

Features include event creation, invitations, notifications, and user-friendly dashboards. The project emphasizes cross-platform compatibility, smooth UI, and responsive design for both Android and iOS devices.
