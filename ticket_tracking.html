Ticket Tracking Web App

Project Information

  • Category: Web Application
  • Client: Self Project
  • Project Date: 2025
  • Tech Stack: Angular, .NET, SQL Server

About the Project

The Ticket Tracking Web App is a web-based solution developed using Angular for the frontend and .NET for the backend. It allows users to create, manage, and track tickets efficiently.

Features include ticket creation, status tracking, priority management, user assignment, and reporting dashboards. The project emphasizes performance, usability, and maintainable architecture.
